# Program Jaringan Syaraf Tiruan Menggunakan Tensorflow
Program dapat memprediksi masukkan gambar berupa tangan yang membentuk gunting, batu, atau kertas.

## Rincian Program

* Dataset berasal dari :  https://dicodingacademy.blob.core.windows.net/picodiploma/ml_pemula_academy/rockpaperscissors.zip
* Dataset dibagi menjadi train set dan validation set
* Menggunakan image data generator
* Akurasi yang didapat diatas > 95 %
